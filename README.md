# claude-plugins

Plugin marketplace for Claude Code by [Bazilio-san](https://github.com/Bazilio-san).

## Available plugins

| Plugin | Version | Description |
|--------|---------|-------------|
| [claude-notification-plugin](https://github.com/Bazilio-san/claude-notification-plugin) | 1.1.75 | Claude Code notifications on task completion: Telegram, Windows toast, sound, and voice |

## Install

Add the marketplace:

```shell
/plugin marketplace add Bazilio-san/claude-plugins
```

Install a plugin:

```shell
/plugin install claude-notification-plugin@bazilio-plugins
```
