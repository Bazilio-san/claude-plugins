# claude-plugins

Plugin marketplace for Claude Code by [Bazilio-san](https://github.com/Bazilio-san).

## Available plugins

| Plugin | Description |
|--------|-------------|
| [claude-notification-plugin](https://github.com/Bazilio-san/claude-notification-plugin) | Telegram, Windows toast, sound, and voice notifications for Claude Code task completion |

## Install

Add the marketplace:

```shell
/plugin marketplace add Bazilio-san/claude-plugins
```

Install a plugin:

```shell
/plugin install claude-notification-plugin@bazilio-plugins
```
